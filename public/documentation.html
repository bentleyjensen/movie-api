Documentation

Retrieve various information on various movies. Like imdb.com, but littler.

  • Return a list of ALL movies to the user
  • Return data (description, genre, director, image URL, whether it's featured or not) about a single movie by title to the user
  • Return data about a genre (description) by name/title (e.g., "Thriller")
  • Return data about a director (bio, birth year, death year) by name
  • Allow new users to register
  • Allow users to update their user info (username, password, email, date of birth)
  • Allow users to add a movie to their list of favorites
  • Allow users to remove a movie from their list of favorites
  • Allow existing users to deregister

GET /movies

Get all movies as JSON. Note that it is an array,
versus directors and genres, which is an object.

Request Body: None

Response: JSON Array containing objects

[
    {
        title: 'Casino Royale',
        year: 2006,
        rating: 'PG-13',
        director: 'Martin Campbell',
        stars: ['Daniel Craig', 'Eva Green', 'Judy Dench'],
        series: '007'
    },
    {
        title: 'Quantum Of Solace',
        year: 2008,
        rating: 'PG-13',
        director: 'Marc Forster',
        stars: ['Daniel Craig', 'Olga Kurylenko', 'Mathieu Amalric'],
        series: '007'
    }
]

GET /movies/title/{title}

Find a specific movie by title

Request Body: None

Response: JSON Object

A GET request to /movies/title/Skyfall would return
a single movie object. If a movie could not be found,
response will be 404 and an empty object

{
    title: 'Skyfall',
    year: 2012,
    rating: 'PG-13',
    genres: ['Action', 'Adventure', 'thriller'],
    director: 'Sam Mendez',
    stars: ['Daniel Craig', 'Javier Bardem', 'Naomie Harris'],
    series: '007'
}

GET /directors/{name}

Find a specific director

Request Body: None

Response: JSON Object

The response is an object whose only key is {name}.
If the director cannot be found, you'll receive a 404
and an empty object.

{
    'Sam Mendez': {
        movies: [
            'Skyfall',
            'Spectre',
            'American Beauty',
            '1917'
        ],
        birthday: '1965-08-01',
        occupations: [
            'Director',
            'Producer',
            'Screenwriter'
        ]
    }
}

GET /genres/{genre}

A specific genre's information

Request Body: None

Response: JSON Object

A JSON object whose only key is {genre}.
If the genre cannot be found, a 404 and empty object are returned.

{
    'short': {
        movies: [
            'For The Birds',
            'Put that thing back where it came from or so help me',
            'Bao',
            'Soul'
        ],
        description: 'Not too long!'
    }
}

GET /users/{user}

A specific users's information

Request Body: None (Authentication required soon)

Response: JSON Object

A JSON object whose only key is {user}.
If the user cannot be found, a 404 and empty object are returned.

{
    'bentley': {
        id: 5,
        favorites: [
            'For The Birds',
            'Skyfall'
        ]
    }
}

POST /users

Create a new user

Request Body: JSON with key 'username' (required) and optional key 'password'

{
    username: 'newUser', // Required
    password: 'NewPassword' // Optional
}

Response: JSON Object

A JSON object representing the new user with its ID.

{
    'newUser': {
        id: 1234,
        password: 'NewPassword',
        favorites: []
    }
}

PATCH /users/{username}

Update a user's username from {username} to name from request body

Request Body: JSON with key 'name' (required)

{
    name: 'newUserName'
}

Response: JSON Object

A JSON object representing the new user with its new username.

{
    'newUserName': {
        id: 1234
        favorites: [
        ]
    }
}

DELETE /users/{userId}

Delete user whose id is {userId}

Request Body: None

Response: String

A plaintext success or failure message is returned.

POST /users/{user}/favorites/{newFavorite}

Add {newFavorite} to {user}'s favorites

Request Body: none (Authentication required soon)

Response: JSON Object

A JSON array of the updated favorites.

POST /users/bentley/favorites/Quantum%20Of%20Solace
[
    'For The Birds',
    'Skyfall',
    'Quantum of Solace'

]

DELETE /users/{user}/favorites/{oldFavorite}

Remove {oldFavorite} from a {user}'s favorites

Request Body: none (Authentication required soon)

Response: JSON Object

A JSON array of the updated favorites.

DELETE /users/bentley/favorites/Skyfall
[
    'For The Birds'
